# HAL-MONI (할머니) – Risk Index

Vite + React + TypeScript + Tailwind app for the HAL-MONI pilot dashboard.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This repo is configured to deploy automatically to **GitHub Pages** using the
workflow in `.github/workflows/deploy.yml`.

In your repository settings:

- Go to **Settings → Pages**
- Set **Source** to **GitHub Actions**

On push to `main`, the app will be built and deployed.
